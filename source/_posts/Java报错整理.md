# 资源包报错

## 使用Junit4.12时报错

**报错信息：**

```java
java.lang.NoClassDefFoundError: org/hamcrest/SelfDescribing
```

**解决方案：**

1. 更换版本为4.10,版本4.11和4.12都不行，如果必须使用4.12，请使用方案2
2. Junit4.12 + hamcrest-core-1.3
